## Requirements
To run this project is necessary:

- skfuzzy
- numpy
- mathplotlib
- re
- generativeai (pip install -q -U google-generativeai)
- Django
- nltk
- - nltk.download('vader_lexicon')  
- - nltk.download('punkt')  
- sentence-transformers
- scipy
- implicit

